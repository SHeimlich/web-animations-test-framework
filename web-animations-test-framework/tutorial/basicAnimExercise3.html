Now you know how to let the animation move left, right, top and down as well as diagonally. How about you try to let the move in a rectangle clockwise. So this is what you need to achieve:

  1. Move right 400px

  2. Move down 200px

  3. Move left 400px

  4. Then move back up 200px

  5. Each move will take 2 seconds to run.


      6. Show Solution
      new Animation(document.querySelector("#a"), {left: "400px"}, 2); new Animation(document.querySelector("#a"), {top: "200px"}, 2); new Animation(document.querySelector("#a"), {left: "0px"}, 2); new Animation(document.querySelector("#a"), {top: "0px"}, 2);

      You might wonder why does the rectangle moves diagonally when you have 2 new Animation() commands. That is because you have made 2 animation objects at the same time and let them ran at the same time, if you want the movement in sequence, then you will have to apply a 'startDelay' to the timing as follow:

      new Animation(document.querySelector("#a"), {left: "300px"}, 2.0); new Animation(document.querySelector("#a"), {top: "500px"}, {startDelay: 2, duration: 2});

      Notice how on the second animation object, we have put 'startDelay' and 'duration' in {}, that is because we are creating a timing dictionary to apply to the animation. Any timing dictionary will have to be stored in {}. You might also notice that number '2' is gone! That is because we have apply number '2' to 'duration' which is the duration of the animation.
