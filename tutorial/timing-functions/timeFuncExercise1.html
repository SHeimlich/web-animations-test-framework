Exercise 1 - Start and End Slowly

It's boring to see animations moving at one speed and to see animations to start and stop in a sudden which is not smooth at all. Now we know that there are several easing effects, let's try one in this exercise.

In this exercise, create an animation that moves 300px from left and has ease-in-out effect applied to it for 5 seconds.

Show Solution
new Animation(document.querySelector("#a"), {left: "300px"}, {duration: 5, timingFunction: new TimingFunction("ease-in-out")});
