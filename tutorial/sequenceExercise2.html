Sequential Animation Group
Exercise 2 - Parent Inside Children

Do you remember what you did in the second exercise for parallel animation group? This exercise will require you to do it the same way but using sequential animation group.

In this exercise, First create the parent sequence group, then include the group into each of the animation.

The same as previous exercise: create 3 animations that runs sequentially, the first animation runs diagonally 300px from left and 300px from top, the second animation runs diagonally 600px from left and 0px from top, then the third animation moves back to the start position. Each animation should take 3 seconds to run. The whole cycle should be repeated twice.

Show Solution
var group = new SeqGroup([], {iterationCount: 2}); new Animation(document.querySelector("#a"), {left: "300px", top: "300px"}, 3, group);
new Animation(document.querySelector("#a"), {left: "600px", top: "0px"}, 3, group);
new Animation(document.querySelector("#a"), {left: "0px", top: "0px"}, 3, group);
